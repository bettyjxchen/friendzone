Web App Connecting Healthcare Institutions

1. Purpose

Develop a platform to administer and connect healthcare institutions to their patients. Allow direct communication and group channel messaging to foster community and mentorship in institutions.

2. Admin Use

Built secure account management and authorization features utilizing password and cookie-hashing and tamperproof HMAC URLs. Developed an administrative data management interface for manually adjusting or removing user accounts and user-generated content.

3. Client Use

Developed a notification system incorporating on-site and email messaging. Created a direct messaging system to allow chat capabilities between users receiving treatment at healthcare locations within the same institution tenant.
Project Details

Client: Laso Concepts

Date: December 2017

Online: http://laso-concepts.herokuapp.com

Laso Concepts is a platform for connecting healthcare institutions, patients, and support teams. The MVP is built as a mobile responsive single page application (SPA).
Features multi-tenancy, thorough user-onboarding, multiple profile support, hierarchical group messaging with keyword search, tagging and mentions, direct messaging, on-site and email notifications, blogging and content management systems, and content moderation.
